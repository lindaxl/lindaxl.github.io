发表于 2018-11-02 | 分类于 HEXO

问题

我们如果换了电脑怎么办？我们把hexo文件从一个电脑cope到另外一个电脑吗?答案肯定不是这样的，因为这里面有好多依赖包，好几万个文件呢，这样显然不合理

解决方案

我们初步的解决方案是把我们的文件提交到git上，利用git来管理它，我是这样解决的： 在现有的XXX.github.io项目上创建一个分支来管理

  1. 克隆gitHub上的lindaxl.github.io项目的文件到本地
    git clone git@github.com:lindaxl/lindaxl.github.io.git
  2. 删除文件夹里除了.git的其他所有文件
  3. 把hexo项目文件下的所有文件全部复制过来
  4. 里面应该有一个叫.gitignore的文件，如果没有就输入 touch .gitignore，创建一个

  5. .gitignore文件里应该是这些内容

  6. 创建一个叫hexo的分支并切换到这个分支上

  7. 提交复制过来的文件到暂存区

  8. 提交

  9. 推送分支到github

到这一步我们就基本上搞定了，以后再跟新了博客就直接在克隆下来的文件里面 git push origin hexo就可以了，hexo的操作跟以前一样不变。
这是在第一次新建分支时候的操作

  1. 今后无论什么时候想要在其他电脑上面用hexo写博客，就直接把创建的分支克隆下来，npm install安装依赖之后就可以用了。
    克隆分支的操作
    git clone -b hexo https://github.com/yourname/lindaxl.github.io.git

cd lindaxl.github.io
安装依赖
npm install

  1. 因为上面创建的是一个名字叫hexo的分支，所以这里-b后面的是hexo，再把后面的gitHub的地址换成你自己的hexo博客的地址就可以了。
  2. 这样做完了以后，每次写完博客发布之后不要忘了还要git push origin hexo把源文件推到分支上。

Javascript的max方法

发表于 2018-03-19 | 分类于 Javascript

var a = [1,4,5,2,9];
求最大值
console.log(Math.max.apply(null,a)) //9 这是使用apply的方法
console.log(Math.max(…a)) //9 这是ES6语法

Math对象包含max()方法，用于确认一组数值中的最大值。该方法接收任意多个数值参数，不接受数组参数。

要找到数组中的最值，可以使用apply()方法，D表示将Math.max()方法的执行环境切换到null上，apply()方法接收两个参数，第二个参数是一个数组。
